Итак,<br/> 
изучим сперва, как взаимосвязаны: поток-давление-сопротивление
</span> 
<br/> 
<br/> 

<span style="font-family: 'Montserrat'; font-size: 1.5em; color: #7b7d7d;">
Поток - это сколько литров (или милилитров, или микролитров) протекает по трубочке за секунду (или минуту, или час). <br/> 
Поток также называют объемной скоростью, но не путаем ее с обычной (линейной) скоростью - сколько метров за секунду. <br/> 
<br/>
Для лучшего понимания разницы между объемной скоростью и линейной скоростью, представим себе огромную реку, которая медленно течет по равнине. Если мы поместим в реку игрушечный кораблик, он будет плыть медленно (преодолевать мало метров в секунду). Следовательно, линейная скорость воды в реке - низкая. А что насчет объемной скорости? Как много литров протекает по реке за секунду? Очень много, так как река очень широкая (у нее большое русло). <br/>
Напротив, по быстрой горной реке протекает гораздо меньший объем (меньше литров за секунду), но вода движется очень быстро. <br/>
<br/>
<br/>
Объемная скорость обозначается буквой Q (иногда V c точекой наверху), единица измерения - л/сек (более строго - кубический метр (1000 л)/сек) <br/>
<br/>
<br/>
Что "заставляет" жидкость двигаться по трубочке? <br/>
<br/>
Разница (градиент) давлений - жидкость движется из области высокого давления в область низкого. <br/>
Давление "толкает"/"выдавливает" жидкость туда, где давление меньше.
</span>  <br/>

<span style="font-family: 'Montserrat'; font-size: 0.7em; color: #7b7d7d;">
(Строго говоря это не так - жидкость двигается из-за градиента энергии, а не градиента давления. Подробнее в первом разделе "Minimum minimorum из физики")
</span> 

<span style="font-family: 'Montserrat'; font-size: 1.5em; color: #7b7d7d;">
Градиент давления обозначается так: P1 минус P2 (или ΔP) <br/> 
P1 - это давление в начале трубочки, P2 - давление в конце трубочки  <br/> 
<br/>
А где начало и конец трубочки в живом организме? Какая трубочка имеется в виду? <br/>
Здесь мы обсуждаем артериальное давление - то есть давление в артериях большого круга  <br/>
Следовательно, здесь нас интересуют его начало и конец: левый желудочек и правое предсердие  <br/>
Р1 - это давление на выходе из левого желудочка/входе в аорту <br/>
То есть - это среднее артериальное давление (почему среднее удобнее использовать, чем систолическое/диастолическое, мы узнаем ниже)<br/>
<br/>
Р2 - это давление в правом предсердии (то есть почти то же самое, что центральное венозное давление (ЦВД))<br/>
